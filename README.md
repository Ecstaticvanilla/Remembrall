## Setup Steps

Install the extension from here  [click me!](https://ecstaticvanilla.github.io/Remembrallwebsite/)
<br>
<img src="https://github.com/user-attachments/assets/0f7cfcb1-29fe-4834-8e32-ad4c13057bfe" width="650">
<br>

<br>
<img src="https://github.com/user-attachments/assets/2b60bb95-e7f9-49df-8003-8b4664bb5430" width="650">
<br>

Open your browser<br>
Go to `Manage Extensions`
<br>
<img src="https://github.com/user-attachments/assets/8ed6d108-cdd8-4b80-9a38-041579a2807a" width="650">
<br>

Enable `Developer Mode`<br>
Click `Load unpacked extension`<br>
Select the extracted extension folder
<br>
